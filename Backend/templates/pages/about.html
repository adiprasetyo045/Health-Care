02. Core Intelligence

Mekanisme Algoritma Decision Tree (CART)

üß† Konsep Recursive Partitioning

Sistem menggunakan algoritma CART (Classification and Regression Tree). Algoritma ini memecah data secara berulang (iteratif) untuk memisahkan kelas Diabetes dan Non-Diabetes dengan memaksimalkan kemurnian data (homogenitas) di setiap simpul anak.

[RUMUS 1] Entropy (H) - Mengukur Ketidakteraturan
H(S) = -ppos √ó log2(ppos) - pneg √ó log2(pneg) // p_pos: Proporsi Diabetes, p_neg: Proporsi Sehat. (0 = Murni, 1 = Acak)
[RUMUS 2] Information Gain (IG) - Penentu Fitur Terbaik
IG(S, A) = H(S) - Œ£ (|Sv| / |S|) √ó H(Sv) // IG menghitung berapa banyak ketidakpastian yang berkurang setelah data dipecah oleh fitur A.

üìù Simulasi Perhitungan Manual

Contoh perhitungan pada Root Node (100 Pasien): 60 Sakit (+), 40 Sehat (-).

Langkah 1: Hitung Entropy Awal
p(+) = 0.6, p(-) = 0.4
H(S) = -(0.6 log2 0.6) - (0.4 log2 0.4)
H(S) = 0.971 (Sangat Tidak Murni)
Langkah 2: Coba Pecah (Glukosa > 140)
- Kiri (<=140): 30 Org (5+, 25-)
- Kanan (>140): 70 Org (55+, 15-)
Hitung Weighted Entropy...
Gain = 0.971 - 0.65 = 0.321 (Signifikan!)

‚úÖ Kesimpulan: "Glukosa" dipilih sebagai pemecah utama karena Gain tinggi.

üöÄ Mengapa Decision Tree?

Interpretability (Transparansi)

Berbeda dengan Neural Networks ("Black Box"), alur keputusan Decision Tree dapat ditelusuri sepenuhnya oleh dokter.

Non-Linearity

Mampu menangkap pola kompleks (misal: Risiko tinggi pada usia muda JIKA gula darah sangat ekstrem).

Robustness

Tidak terpengaruh signifikan oleh outlier data klinis yang ekstrem.

03. Scoring System

Perhitungan Probabilitas & Risiko

üå°Ô∏è Dari Klasifikasi ke Persentase

Standard Decision Tree hanya memberikan output 0 (Sehat) atau 1 (Sakit). Untuk kebutuhan medis, kami menerapkan Calibrated Classifier (Isotonic Regression).

Probabilitas(Positif) = Jumlah Sampel Positif di Daun / Total Sampel di Daun

Sistem menghitung rasio kelas di "daun" (leaf node) terakhir.
Contoh: Jika sebuah simpul akhir berisi 100 pasien dengan karakteristik serupa, dan 85 di antaranya positif diabetes, maka Confidence Score = 85%.

üìä Interpretasi Level Risiko

Skor probabilitas dipetakan ke dalam kategori risiko klinis untuk memudahkan triase:

0 - 20% Sangat Rendah
21 - 40% Rendah
41 - 60% Sedang (Moderate)
61 - 100% Tinggi / Kritis
